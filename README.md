# Power BI Performance Dashboard

This repository contains an **interactive Performance Dashboard built using Power BI**, designed to track revenue, profit, quantity, customers, products, and geographic performance over time.

The dashboard enables stakeholders to monitor key business KPIs and identify trends across products, customers, channels, and cities.
